Help for PlainZip users.

PlainZip is distributed through Microsoft Store and runs archive preview, extraction, and creation locally on your Windows device.

Open Microsoft Store listing Privacy policy

Install And Update

Install PlainZip and receive updates from the Microsoft Store product page.

  • Use the Store listing to install the latest available version.
  • If an update is not visible yet, check again after the Store finishes rollout for your device and region.
  • PlainZip does not require a separate 7-Zip installation.

Common Questions

Where are my files processed?

Archive operations run locally. PlainZip does not upload your archive contents.

Why was an archive skipped?

Password-required or high-risk archives may need review before extraction.

Can I check privacy details?

The privacy policy explains file access, local settings, logs, and password handling.

Troubleshooting

  • If an archive does not open, confirm the file exists locally and is not still downloading.
  • If extraction fails, try the archive integrity test before extracting again.
  • If an encrypted archive asks for a password, PlainZip uses the password only for the active operation.
  • If a Store update appears delayed, open the Microsoft Store library and check for app updates.

Contact Path

For product support, use the support or contact option available from the PlainZip Microsoft Store listing.
